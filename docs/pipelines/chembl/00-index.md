# ChEMBL Pipelines

Набор ETL-пайплайнов для извлечения данных из базы ChEMBL через REST API.

## Обзор
Все пайплайны этого семейства наследуют общую логику из `ChemblCommonPipeline`, что обеспечивает единый подход к конфигурации, пагинации и валидации.

## Доступные пайплайны

| Pipeline Name | Сущность | Endpoint | Описание |
|---------------|----------|----------|----------|
| `activity_chembl` | **Activity** | `/activity` | Результаты измерений биоактивности. Самая объемная сущность. |
| `assay_chembl` | **Assay** | `/assay` | Описания экспериментов. |
| `target_chembl` | **Target** | `/target` | Биологические мишени. |
| `testitem_chembl` | **TestItem** | `/molecule` | Тестируемые химические соединения (молекулы). |
| `document_chembl` | **Document** | `/document` | Научные публикации (статьи, патенты). Поддерживает внешнее обогащение. |
