# 00 Index

## Конфигурационная подсистема
- **ConfigResolver** — загружает и объединяет YAML-профили (base/dev/prod), применяет приоритеты и расширения.
- **EnvSecretProvider** — подставляет секреты из переменных окружения.
- **SecretProviderABC** — контракт для любых поставщиков секретов.

## Профили и приоритеты
- Конфиги организованы по профилям; dev наследует base, prod усиливает ограничения и логирование.
- Переменные окружения имеют приоритет над YAML значениями для ключей, помеченных как секреты.

## Применение в пайплайнах
Пайплайн читает конфиг через ConfigResolver, получает параметры клиента (пагинация, лимиты), настройки детерминизма и пути вывода. См. примеры в `docs/guides/01-configuration.md`.
