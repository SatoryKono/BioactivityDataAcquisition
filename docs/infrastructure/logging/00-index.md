# 00 Index

## Компоненты логирования
- **UnifiedLogger** — фасад структурированного логирования с контекстом pipeline/run.
- **LoggerAdapterABC** — адаптер под конкретные логгер-бэкенды.
- **ProgressReporterABC** — отчёты о прогрессе (страницы API, количество записей).
- **TracerABC** — трассировка шагов и метрик производительности.

## Принципы
- Везде передаются `pipeline_name` и `run_id` для корреляции событий.
- Сообщения структурированы (ключ-значение) и готовы к агрегации.

## Интеграция
- PipelineBase использует UnifiedLogger в каждой стадии.
- Клиенты и пагинаторы публикуют прогресс и ошибки через ProgressReporter/Tracer.
