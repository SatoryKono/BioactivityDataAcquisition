# Document Term (`document_term`)

| Поле | Тип данных | Допустимые значения | Описание |
|------|------------|----------------------|----------|
| document_chembl_id | строка | — | Документ, для которого извлечены ключевые термины. |
| term_text | строка | — | Ключевой термин/фраза, найденные TextRank. |
| score | десятичное | — | Вес/важность термина для данного документа. |
