# 00 Rest Api Sources

## Поддерживаемые источники
- **ChEMBL** — основная база биоактивностей, ассеев, мишеней, тест-объектов и документов.
- **Semantic Scholar** — аннотации и цитирования научных статей.
- **PubMed** — метаданные медицинских публикаций (PMID, абстракты).
- **CrossRef** — разрешение DOI и дополнительные библиографические данные.
- **PubChem** — химические идентификаторы и свойства соединений.
- **UniProt** — белковые последовательности и аннотации мишеней.
- **IUPHAR/Guide to Pharmacology** — данные об интеракциях и классификациях таргетов.
- **OpenAlex** — открытые метаданные публикаций и связей.

## Роли в графе данных
ChEMBL даёт основную фактуру данных; PubChem и UniProt/IUPHAR используются для обогащения TestItem и Target; PubMed/CrossRef/Semantic Scholar/OpenAlex — для публикационных метаданных и связей между экспериментами.

## Общие ограничения и форматы
Источники предоставляют REST/JSON API с лимитами RPS; клиенты используют rate limiting, ретраи и кэширование. Основные сущности: активности, ассеи, мишени, соединения, документы; ответы нормализуются в ABC-модель проекта.
