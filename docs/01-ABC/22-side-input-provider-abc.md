# 22 Side Input Provider ABC

## Описание

SideInputProviderABC предоставляет доступ к сайд-инпутам, используемым при обогащении и нормализации данных. Под сайд-инпутами понимаются внешние словари, справочники, конфигурационные маппинги и другие структуры, на которые опираются трансформации. Интерфейс возвращает Mapping[SideKeyT, SideValueT], не навязывая конкретного способа хранения и загрузки: это может быть кэш в памяти, чтение из файла, запрос к БД или HTTP-сервису. Реализация должна явно управлять жизненным циклом ресурсов (подключения, файловые дескрипторы) и гарантировать консистентность предоставляемых данных в рамках одного запуска пайплайна. SideInputProviderABC не должен заниматься логикой трансформаций самих записей и не должен вшивать бизнес-правила в процесс загрузки; его задача — стабильно и предсказуемо отдавать вспомогательную информацию, на которую будут опираться трансформеры и обогащатели.

## Interface

```python
from typing import Generic, TypeVar, Mapping

SideKeyT = TypeVar("SideKeyT")
SideValueT = TypeVar("SideValueT")

class SideInputProviderABC(Generic[SideKeyT, SideValueT]):
    def get_side_inputs(self) -> Mapping[SideKeyT, SideValueT]:
        """Получить сайд-инпуты для обогащения."""
        raise NotImplementedError
```
