# 21 Secret Provider ABC

## Описание

SecretProviderABC предоставляет доступ к секретным данным (API ключи, токены, пароли). Он инкапсулирует логику получения секретов из различных источников (переменные окружения, секрет-менеджеры, файлы) и обеспечивает единообразный интерфейс доступа. Интерфейс не навязывает конкретный способ хранения секретов, но гарантирует безопасное получение значений по имени. Реализация должна следовать принципам безопасности: не логировать секреты, не хранить их в открытом виде, поддерживать ротацию.

## Interface

```python
from abc import ABC, abstractmethod

class SecretProviderABC(ABC):
    """Доступ к секретным данным."""
    
    @abstractmethod
    def get_secret(self, name: str) -> str:
        """Получить секрет по имени."""
        pass
```

## Related Components

- **SourceClientABC**: может использовать secret provider для аутентификации
- **RequestBuilderABC**: может использовать secret provider для добавления заголовков
