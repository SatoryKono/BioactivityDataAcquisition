# 16 Rate Limiter ABC

## Описание

RateLimiterABC определяет интерфейс для локального ограничения частоты запросов и параллелизма. Он предоставляет методы для получения и освобождения слотов, что позволяет контролировать нагрузку на внешние API и предотвращать превышение лимитов. Реализация может использовать различные алгоритмы (token bucket, sliding window, semaphore) и настраиваемые параметры (QPS, количество параллельных запросов). Интерфейс не навязывает конкретный алгоритм, но гарантирует блокирующее поведение при отсутствии доступных слотов.

## Interface

```python
class RateLimiterABC:
    def acquire(self) -> None:
        """Заблокировать выполнение до получения слота для операции."""
        raise NotImplementedError

    def release(self) -> None:
        """Освободить ранее занятый слот."""
        raise NotImplementedError
```
