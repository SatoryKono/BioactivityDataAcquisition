# 16 Rate Limiter ABC

## Описание

RateLimiterABC определяет интерфейс ограничителя частоты запросов. Обеспечивает соблюдение лимитов нагрузки при обращении к внешнему API (например, не более N запросов в секунду), чтобы избежать превышения квот или блокировок. Он предоставляет методы для получения и освобождения слотов, что позволяет контролировать нагрузку на внешние API и предотвращать превышение лимитов. Реализация может использовать различные алгоритмы (token bucket, sliding window, semaphore) и настраиваемые параметры (QPS, количество параллельных запросов). Метод `acquire()` блокирует выполнение до получения слота для операции, приостанавливая выполнение пока не наступит момент, когда можно выполнить запрос согласно заданному лимиту (например, не более N запросов в окне времени). Интерфейс не навязывает конкретный алгоритм, но гарантирует блокирующее поведение при отсутствии доступных слотов.

## Interface

```python
class RateLimiterABC:
    def acquire(self) -> None:
        """Заблокировать выполнение до получения слота для операции."""
        raise NotImplementedError

    def release(self) -> None:
        """Освободить ранее занятый слот."""
        raise NotImplementedError
```
