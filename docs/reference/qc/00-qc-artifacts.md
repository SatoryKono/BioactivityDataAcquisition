# 00 Qc Artifacts

## meta.yaml
Содержит `pipeline_name`, `pipeline_version`, `chembl_release`, `run_id`, `row_count`, контрольные суммы таблиц и пути QC-файлов. Используется для воспроизводимости и аудита.

## QC-таблицы
- `quality_report_table.csv` — агрегированные проверки полноты, типов и распределений.
- `correlation_report_table.csv` — корреляции и сравнение с предыдущими сборками при наличии эталонов.

## Checksums
Файлы в `checksums/` (например, `activity_table.sha256`) фиксируют контрольные суммы таблиц и отчётов, обеспечивая детерминизм и возможность быстрых регресс-тестов.
