# 00 Cli Overview

## Архитектура CLI
- Основана на Typer (или совместимой библиотеке) с декларативным описанием команд.
- Базовый контракт: `CLICommandABC` определяет регистрацию и исполнение команд.
- Команды расширяемы: новые пайплайны добавляют собственные команды, регистрируя их в общем приложении.

## Связь с PipelineBase
CLI команды вызывают методы PipelineBase через слой оркестрации, передавая конфиг, профиль и параметры вывода. Конфигурации читаются через ConfigResolver.

## Расширяемость
Команды организованы модульно: можно добавить новую команду в отдельном модуле и зарегистрировать её в точке входа CLI без изменения существующих команд.
