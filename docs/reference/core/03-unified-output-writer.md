# 03 Unified Output Writer

## Роль
`UnifiedOutputWriter` обеспечивает единый механизм записи всех артефактов пайплайна с атомарностью и детерминированным порядком.

## Связь с ABC
- Опирается на `WriterABC` для таблиц (CSV/Parquet) и `MetadataWriterABC` для `meta.yaml`.
- Управляет планированием путей и проверкой существования временных файлов перед заменой.

## Атомарная запись
- Создаёт временные файлы, затем использует `os.replace` для финализации.
- Гарантирует фиксированный порядок колонок и сортировку строк перед записью.

## Артефакты
- Таблицы в каталоге `tables/`.
- `meta.yaml` с параметрами запуска, версиями релизов, количественными метриками и checksums.
- QC-отчёты (`quality_report_table.csv`, `correlation_report_table.csv`) и файлы контрольных сумм в `checksums/`.
