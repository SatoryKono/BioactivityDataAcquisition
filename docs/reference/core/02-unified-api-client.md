# 02 Unified Api Client

## Основные компоненты
- **UnifiedAPIClient** — фасад, предоставляющий единый интерфейс для вызовов внешних API и применения инфраструктурных политик.
- **BaseClient** — абстракция конкретного источника, реализующая доменные методы извлечения.
- **ConfiguredHttpClient** — настройка HTTP-бэкенда с лимитами, ретраями, кэшем и circuit breaker.

## Надёжность и ограничения
- Ретраи с экспоненциальным backoff, управление скоростью через TokenBucketRateLimiter.
- Circuit breaker для отсечки нестабильных источников.
- Кэширование ответов (файловое или в памяти) для детерминизма и снижения нагрузки.

## Контракт работы
- Клиент получает RequestBuilder/Parser/Paginator и выполняет цикл извлечения.
- UnifiedAPIClient управляет HTTP-бэкендом и передаёт результаты в трансформацию пайплайна.
- Примеры реализаций: ChemblClient и другие клиенты REST-источников.
