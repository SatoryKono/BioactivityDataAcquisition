# 04 Schema Registry

## Назначение
`SchemaRegistry` управляет регистрацией и выдачей Pandera-схем для доменных сущностей, обеспечивая единый источник истины для валидации и сортировки колонок.

## Работа с SchemaProviderABC
- `SchemaProviderABC` описывает контракт поставщика схем (получение по имени сущности/версии).
- Регистрация схем включает `column_order` для детерминированного вывода и определения бизнес-ключей.

## Использование в пайплайнах
- PipelineBase запрашивает схему у реестра перед валидацией.
- Колонки упорядочиваются согласно `column_order`, затем вычисляются хеши (`hash_business_key`, `hash_row`).
- При несовпадении колонок/типов ValidationService сигнализирует об ошибке валидации.
