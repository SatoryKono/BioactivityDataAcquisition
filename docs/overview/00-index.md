# 00 Index

## Назначение проекта
BioETL — это модульная ETL-система для извлечения, нормализации и интеграции данных о биоактивности из открытых источников. Цель проекта — собрать сопоставимые Assay/Activity/Target/TestItem данные, сохранить детерминизм сборки и предоставить прозрачные артефакты контроля качества.

## ABC-модель доменных сущностей
- **Assay** — описание эксперимента: условия, метод, биоматериал и связка с источником данных.
- **Activity** — измеренный результат биоактивности (значения, единицы, контекст ассея) с бизнес-ключом и хешами для детерминизма.
- **Target** — биологическая мишень (белок/комплекс), к которой относится эксперимент или активность.
- **TestItem** — тестируемый объект (соединение, препарат, антитело) с идентификаторами источников.

## Ключевые внешние источники данных
- ChEMBL — основной источник для Activity, Assay, Target, TestItem, Document.
- PubMed, PubChem — публикации и химические идентификаторы.
- UniProt, IUPHAR/Guide to Pharmacology — мишени и их аннотации.
- CrossRef, Semantic Scholar, OpenAlex — библиографические метаданные и цитирования.
