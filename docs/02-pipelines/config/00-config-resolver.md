# 00 Config Resolver

## Описание

`FileConfigResolver` — реализация резолвера конфигурации пайплайна из YAML-файлов. Загружает основной конфиг и профильные слоя, сливает их и применяет переопределения окружения и переменных для получения окончательного `PipelineConfig`.

## Модуль

`src/bioetl/core/config/config_resolver.py`

## Наследование

Резолвер реализует интерфейс `ConfigResolverABC` и предоставляет реализацию загрузки конфигурации из файлов.

## Основной метод

### `resolve(self, profile: str | Path | None = None, overrides: Mapping[str, Any] | None = None) -> PipelineConfig`

Разрешает конфигурацию пайплайна из файлов с учётом профиля и переопределений.

**Параметры:**
- `profile` — имя профиля или путь к файлу конфигурации (опционально)
- `overrides` — переопределения конфигурации (опционально)

**Процесс разрешения:**

1. Загрузка основного конфига: чтение базового YAML-файла конфигурации
2. Загрузка профиля: применение профильного слоя конфигурации (если указан)
3. Слияние конфигов: объединение базового конфига и профиля
4. Применение переопределений: применение переопределений из `overrides`
5. Замена переменных: подстановка переменных окружения
6. Валидация: проверка корректности финальной конфигурации

**Возвращает:** объект `PipelineConfig` с финальной конфигурацией.

## Профили конфигурации

Резолвер поддерживает профили для различных окружений:
- `development` — разработка
- `testing` — тестирование
- `production` — продакшн

## Переопределения

Переопределения позволяют изменять конфигурацию без редактирования файлов:
- Через параметр `overrides`
- Через переменные окружения
- Через CLI-параметры
