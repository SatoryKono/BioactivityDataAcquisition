# 02 TestItem ChEMBL Thin Pipeline

## Описание

`ChemblTestItemThinPipeline` — "тонкий" вариант ChEMBL-пайплайна для тестовых элементов. Минимальная реализация без PubChem-обогащения, включающая только базовые поля и сортировку.

## Модуль

`src/bioetl/pipelines/chembl/thin/run.py`

## Наследование

Пайплайн наследуется от `ChemblPipeline` и предоставляет упрощённую реализацию для случаев, когда обогащение через PubChem не требуется.

## Основные методы

### `__init__(self, ...)`

Инициализирует "тонкий" пайплайн с минимальной конфигурацией.

**Особенности:**
- Не требует настройки PubChem клиента
- Использует только базовые поля из ChEMBL
- Минимальная обработка данных

## Использование

"Тонкий" пайплайн используется в случаях:

- Быстрой выгрузки базовых данных без обогащения
- Тестирования инфраструктуры пайплайнов
- Ситуаций, когда PubChem недоступен или не требуется

## Отличия от TestItemChemblPipeline

- Отсутствует обогащение через PubChem
- Минимальная трансформация данных
- Быстрее выполняется за счёт отсутствия внешних запросов

## Related Components

- **ChemblPipeline**: базовый класс ChEMBL-пайплайнов
