# 01 Target Normalizer

## Описание

`TargetNormalizer` — класс-заглушка для нормализации данных *target* на основе контролируемых словарей (справочников). Предполагается для обработки и стандартизации значений (например, типов таргетов, организмов и т.п.) после извлечения и перед загрузкой, однако текущая реализация не заполнена (метод вызывает NotImplementedError).

## Модуль

`src/bioetl/pipelines/chembl/target/normalizers.py`

## Основные методы

### `normalize(self, df) -> pd.DataFrame`

Должен выполнить нормализацию DataFrame (например, привести типы к строкам, заполнить пропуски, вычислить бизнес-ключи, хэши и т.д.), но в текущей заглушке просто бросает исключение NotImplementedError.

**Параметры:**
- `df` — DataFrame для нормализации

**Возвращает:** нормализованный DataFrame (в текущей реализации выбрасывает исключение).

## Использование

Нормализатор предназначен для использования в пайплайне target, но в текущей версии не реализован.
